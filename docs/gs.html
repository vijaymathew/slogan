Getting started with Slogan

The latest development code base of Slogan can be obtained from GitHub:

$ git clone https://github.com/vijaymathew/slogan.git

After cloning the repository, use these commands to compile and install Slogan:

$ cd slogan
$ ./configure-platform
$ make
$ sudo make install
$ slogan

The last command will launch the Slogan REPL:

slogan>
./index
