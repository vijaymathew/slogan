subs64array_fill(a, start, end, fill)

Similar to u8/s8/u16/s16/u32/s32/u64/s64array_fill, but only the elements from start (inclusive) to end (exclusive) are replaced with fill.

Return void.

Examples:


let a = #u8[1, 2, 3, 4, 5]
subu8array_fill(a, 2, 4, 10)
a
// #u8[1, 2, 10, 10, 5]
