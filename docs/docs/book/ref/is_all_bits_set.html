is_all_bits_set


is_all_bits_set(i1, i2)

Return true if the bitwise and of the integers i1 and i2 is equal to i1, otherwise return false.

Examples:


is_all_bits_set(0b0010, 0b0010)
// true

is_all_bits_set(0b0010, 0b0100)
// false
