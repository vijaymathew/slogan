error


error(message, obj1, ...)

Raises an error-exception object whose message field is message and arguments field is the list of values obj1, ....

Return void.

Examples:


try
  error("connection failed", "www.abcd.com", 80)
catch (e)
{ showln(error_message(e))
  showln(error_args(e)) }

//> connection failed
    [www.abcd.com, 80]
