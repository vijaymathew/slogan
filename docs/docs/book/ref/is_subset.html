is_subset


is_subset(s1, s2)

Return true if the set s1 is a subset of set s2.

Otherwise return false.

Examples:


is_subset(#(1,2), #(2))
// false
is_subset(#(1,2), #(1,2))
// true
is_subset(#(1,2,3), #(1,2))
// false
is_subset(#(3), #(1,2))
// false
is_subset(#(2), #(1,2))
// true
