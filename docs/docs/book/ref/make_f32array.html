make_f32/f64array


make_f32array(n, @optional fill = 0.0)
make_f64array(n, @optional fill = 0.0)

Return a f32/f64array of size n. Each element will be initialized to the value of fill.

Examples:


make_f32array(3)
// #f32[0., 0., 0.]
make_f32array(3, 1.2)
// #f32[1.2000000476837158, 1.2000000476837158, 1.2000000476837158]
make_f64array(3, 1.2)
// #f64[1.2, 1.2, 1.2]
