time


time(expr)

Return the result of evaluating the expression expr.

As a side effect time displays a message on the standard output which indicates how long the evaluation took (in real time and cpu time), how much time was spent in the garbage collector, how much memory was allocated during the evaluation and how many minor and major page faults occurred.

Time is a special function implemented in the compiler, and cannot be rebound to a new object.
