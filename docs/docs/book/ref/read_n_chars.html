read_n_chars


read_n_chars(n, s = current_reader())

Return a string containing the next n characters from textual input stream s. Return the eof object if the stream has no more characters.

Examples:


let sr = string_reader("hello, world")
read_char(sr)
// \h
read_n_chars(2, sr)
// el
