f32/f64array_shrink


f32array_shrink(fs, n)
f64array_shrink(fs, n)

Reduce the size of the f32/f64 array to n. Return void.

Examples:


let fs = #f32[100.12, 34.54, 567.322]
f32array_shrink(fs, 1)
fs
// #f32[100.12]
