bit_array_set


bit_array_set(b, n)

Turn the nth bit on in the bit array b. Return void.

Examples:


let b = bit_array(1, 0, 0, 1, 1)
b          
// #b[1, 0, 0, 1, 1]

bit_array_set(b, 1)
b
// #b[1, 1, 0, 1, 1]

b[2] = true
b
// #b[1, 1, 1, 1, 1]
