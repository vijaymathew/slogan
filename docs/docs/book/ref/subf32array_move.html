subf32/f64array_move


subf32/f64array_move(a, start, end, dst_array, dst_start)

Replace part of the contents of the f32/f64array dst_array with part of the contents of the f32/f64array a. Copy elements from a, beginning with index start (inclusive) and ending with index end (exclusive) to dst_array beginning with index dst_start (inclusive).

Return void.

Examples:


let a = #f64[1., 2., 3., 4., 5.]
let b = #f64[10., 20., 30., 40., 50.]
subf64array_move(a, 3, 5, b, 1)
b
// #[10., 4., 5., 40., 50.]
