partial


partial(fn, @restargs)

Return a function pf that applies the function fn to args and any more arguments passed to pf.

Examples:


function mult_3(a, b, c) a * b * c
let mult_2_with_9 = partial(mult_3, 9)
mult_2_with_9(2, 3)
// 54

let mult_with_18 = partial(mult_2_with_9, 2)
mult_with_18(10)
// 180
