list_to_f32/f64array


list_to_f32array(xs)
list_to_f64array(xs)

Return an f32/f64array with all the elements of the list xs. The list must contain only real values.

Examples:


list_to_f32array([1., 2., 3.])
// #f32[1., 2., 3.]
list_to_f64array([1., 2., 3.])
// #f64[1., 2., 3.]
