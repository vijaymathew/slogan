arrays_at


arrays_at(xs, i)

Return a list of elements at the (zero-based) index i of the arrays in the list xs. The index can also be a list of integers that indicates the position inside multi-dimensional arrays.

Examples:


arrays_at([#[1, 2, 3], #['a, 'b, 'c]], 2)
// [3, c]
arrays_at([#[#[1, 2], 3], #[#['a], 'b, 'c]], [0, 0])
// [1, a]
