gensym


gensym(prefix = 'g)

Return a new uninterned symbol.

The optional argument prefix is a symbol that will be used as the prefix for the generated symbol.

Uninterned symbols are guaranteed to be distinct from the symbols created in program source code or generated by functions like read and string_to_symbol.

Examples:


write(gensym())
//> #:g0
write(gensym('me))
//> #:me1
