Return a u8/s8/u16/s16/u32/s32/u64/s64array of size n. Each element will be initialized to the value of fill.

Examples:


make_u8array(3)
// #u8[0, 0, 0]
make_u8array(3, 12)
// #u8[12, 12, 12]
