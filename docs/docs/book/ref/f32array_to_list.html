f32/f64array_to_list


f32array_to_list(fs)
f64array_to_list(fs)

Return a list with the elements of the f32/f64 array fs.

Examples:


let fs = #f64[100.12, 34.54, 567.322]
f64array_to_list(fs)
fs
// [100.12, 34.54, 567.322]
