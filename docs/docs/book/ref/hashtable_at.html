hashtable_at


hashtable_at(h, key, default = false)

Return the value associated with key in the hash table h. Return the value of default if no association is found.

Key should be of the appropriate type expected by the hash and equivalence functions specified when creating the hash table. Default may be any object.

Examples:


let h = #{1:2, 3:4}
hashtable_at(h, 1)
// 2
h[3]
// 4
hashtable_at(h, 10, 123)
// 123
