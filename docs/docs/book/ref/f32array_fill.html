f32/f64array_fill


f32array_fill(fs, f)
f64array_fill(fs, f)

Set all elements in the f32/f64array to the value f. Return void.

Examples:


let fs = #f32[10.21, 45.62]           
f32array_fill(fs, 100.00)
fs
// #f32[100.00, 100.00]
