f32/f64array_set


f32array_set(fs, i, f)
f64array_set(fs, i, f)

Set the ith element in the f32/f64 array fs to f.

Examples:


let fs = #f32[10.22, 45.322, 14.56]
f32array_set(fs, 1, 100.)
fs
// #f32[10.22, 100., 14.56]
fs[2] = 200.
fs
// #f32[10.22, 100., 200.]
