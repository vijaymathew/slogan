char_is_ci_gteq(c1, @rest ...)

Predicates for comparing characters. Return true if the relation holds, otherwise return false.

The char_is_ci_*** predicates ignores the case of the characters being compared.

Examples:


char_is_eq(\a, \A)
// false
char_is_ci_eq(\a, \A)
// true
