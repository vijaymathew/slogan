println


println(@key stream = current_writer(), obj1, ...)
print(@key stream = current_writer(), obj1, ...)

Write the compiled (or internal) representation of the objects obj1, ... to the output-stream stream. Print strings, symbols and characters without quotes. Elements of compound objects (pair, list, array etc) are recursively written without the surrounding tokens (parentheses, spaces, dots, etc).

Println will emit a newline after printing the objects.

Both functions return void.

Examples:


println("it's ", \a, \space, 'wonderful, " world")
//> it's a wonderful world
println([1,2,3])
//> 123
