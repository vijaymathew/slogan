div


div(n1, n2, ...)

Return the result of dividing num1 by the product of num2, ..... The division operator (/) is internally mapped to this function.

Examples:


div(12, 100, 34)
// 3/850
sub(12)
// 1/12
12 / 100 / 34
// 3/850
`/`(12, 100, 34)
// 3/850
