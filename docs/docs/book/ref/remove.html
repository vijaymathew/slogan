remq/remv/remove


remq(obj, xs)
remv(obj, xs) 
remove(obj, xs)

Return a list containing the elements of xs with all occurrences of obj removed. The equivalence test for remq is is_eq, for remv is is_eqv, and for remove is is_equal.

Examples:


remq(1, [1, 2, 1, 3])
// [2, 3]

remv(1020032003209020020020, [1020032003209020020020, 2, 1020032003209020020020, 3])
// [2, 3]

remove([1,2,3], [[1,2,3], [4,5,6], [1,2,3]])
// [[4, 5, 6]]
