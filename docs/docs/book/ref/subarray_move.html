subarray_move


subarray_move(a, start, end, dst_array, dst_start)

Replace part of the contents of the array dst_array with part of the contents of the array a. Copy elements from a, beginning with index start (inclusive) and ending with index end (exclusive) to dst_array beginning with index dst_start (inclusive).

Return void.

Examples:


let a = #[1, 2, 3, 4, 5]
let b = #['a, 'b, 'c, 'd, 'e]
subarray_move(a, 3, 5, b, 1)
b
// #[a, 4, 5, d, e]
