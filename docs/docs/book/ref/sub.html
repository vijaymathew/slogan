sub


sub(n1, n2, ...)

Return the difference between num1 and the sum of num2, ..... The subtraction operator (-) is internally mapped to this function.

Examples:


sub(12, 100, 34)
// -122
sub(12)
// -12
12 - 100 - 34
// -122
`-`(12, 100, 34)
// -122
