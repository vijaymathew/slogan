subarray


subarray(a, start, end)

Return a new array with the elements of the array a from index start (inclusive) to end (exclusive).

Examples:


subarray(#[1, 2, 3, 4, 5], 2, 4)
// #[3, 4]
subarray(#[1, 2, 3, 4, 5], 2, 5)
// #[3, 4, 5]
