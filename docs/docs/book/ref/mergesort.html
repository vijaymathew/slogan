merge/quicksort


mergesort(xs, @optional test = <)
quicksort(xs, @optional test = <)

Return a list containing the elements of the list xs sorted according to test.

Examples:


quicksort([\b, \y, \s])
// [\b, \s, \y]
mergesort([\b, \y, \s], char_is_gt)
// [\y, \s, \b]
