make_array


make_array(d, @optionalfill = false)

Return an array of dimension d.

The dimension d can be either an exact nonnegative integer or a list of exact nonnegative integers. All the elements of the array are initialized to the value of fill.

Examples:


make_array(5) 
// #[false, false, false, false, false]

make_array(5, 'hello)
// #[hello, hello, hello, hello, hello]

make_array([2, 3], 'hello)
// #[#[hello, hello, hello], #[hello, hello, hello]]
