bshift


bshift(i, n)

Return the result of arithmetically shifting the integer i by n bits.

If n is negative, the bits are shifted right. Otherwise, the bits are shifted left.

Examples:


bshift(23, 1)
// 46
bshift(-105, -1)
// -53
