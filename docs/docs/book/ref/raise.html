raise


raise(obj)

Tail-calls the current exception-handler (i.e, the currently active catch expression) with obj as the sole argument. If raise was called from a trycc expression and if the exception-handler returns, the continuation of the call to raise is invoked.

Return void.

Examples:


try 100 + raise('oops) 
catch (e) { showln(e); 2 }
//> oops
// 2

trycc 100 + raise('oops) 
catch (e) { showln(e); 2 }
//> oops
// 102
