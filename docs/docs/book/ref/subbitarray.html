subbitarray


subbitarray(ba, start, end)

Return a new bit array with elements from the bit array ba starting at index start (inclusive) and ending at index end (exclusive).

Examples:


subbitarray(#b[1, 0, 0, 1, 1, 1], 3, 6)
// #b[1, 1, 1]
