member


member(x, xs)

Return the first tail of the list xs whose head is equivalent to x. Return false if x is not found in xs.

The equivalence test for member is is_equal.

Examples:


member("hi", [1, 2, "hi", 4, 5])
// ["hi", 4, 5]

member(1020032003209020020020, [1, 2, 1020032003209020020020, 4, 5])
// [1020032003209020020020, 4, 5]
