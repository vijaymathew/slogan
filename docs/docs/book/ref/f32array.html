f32/f64array


f32array(@rest f1, ...)
f64array(@rest f1, ...)

Return an array of 32-bit or 64-bit floating-point numbers.

32-bit and 64-bit array literals have the #f32 and #f64 prefixes respectively.

Examples:


f32array(1., 2., 3.) == #f32[1., 2., 3.];
// true
f64array(1., 2., 3.) == #f64[1., 2., 3.];
// true
