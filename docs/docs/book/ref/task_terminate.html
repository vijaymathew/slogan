task_terminate


task_terminate(t)

Causes an abnormal termination of the task t. If t is not already terminated, all mutexes owned by it become unlocked/abandoned and a terminated-exception object is stored in its end-exception field. If t is the current task, task_terminate does not return. The actual termination of t will occur at some point between the calling of task_terminate and a finite time in the future.
