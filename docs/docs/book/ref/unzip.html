unzip


unzip(pairs)

Return a pair of lists.

The head of the result will have all heads from pairs and its tail will contain all tails from pairs.

Examples:


let ys = unzip([1:"one", 2:"two", 3:"three"])
head(ys)
// [1, 2, 3]
tail(ys)
// ["one", "two", "three"]
