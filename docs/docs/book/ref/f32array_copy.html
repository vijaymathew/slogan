f32/f64array_copy


f32array_copy(fs)
f64array_copy(fs)

Return an new array of 32-bit or 64-bit floating-point numbers with the same elements in the f32/64array fs.

Examples:


f32array_copy(#f32[1., 2., 3.])
// #f32[1., 2., 3]
