is_empty


is_empty(xs)

Return true if xs is the empty sequence, return false if xs is anything else.

An empty sequence is represented by either the empty list ([]) or false.

Examples:


is_empty([1, 2, 3])
// false

is_empty([])
// true

is_empty("hello")
// false

is_empty(false)
// true
