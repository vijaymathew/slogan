hashtable_contains


hashtable_contains(ht, k)

Return true if an association for the key k exists in hash table ht, otherwise return false.

Examples:


let ht = #{1:100, 2:200}
hashtable_contains(ht, 2)
// true
hashtable_contains(ht, 200)
// false
