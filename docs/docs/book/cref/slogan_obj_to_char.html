___slogan_obj_to_void_pointer(s, t)

These are C macros defined in slogan.h. The ___slogan_obj object s is converted to a C int, unsigned int, float, double or char* and stored in t which should be the address of a C variable of the appropriate type.

Please see ffi_open for example code.
