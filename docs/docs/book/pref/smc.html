package: smc

The smc package implement abstractions that make it easy to work with shared-memory in a concurrent setting.

Installing

The smc package is located under the pkgsrc directory in the Slogan source tree. You can install it from the REPL using the install_package function:


install_package("smc", 'local, "/path_to_slogan_src/pkgsrc/smc")

Once installed, the package can be loaded to a program by calling the load_package function.
